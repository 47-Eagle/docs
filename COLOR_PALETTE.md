# 3D Visualization Color Palette

**Modern & Elegant Design with Eagle Golden Accents**

---

## Color Philosophy

The palette combines:
1. **Eagle Golden** - Your signature brand color (strategic use)
2. **Elegant Indigo** - Modern, sophisticated, professional
3. **Royal Purple** - Premium, elegant complement
4. **Neutral Grays** - Clean, modern foundation

This creates a **premium, modern look** while maintaining Eagle brand identity.

## UI Elements

### Cards & Containers
- Background: `rgba(10, 10, 10, 0.6)` - Deep dark with transparency
- Border: `rgba(212, 175, 55, 0.3)` - Subtle golden accent
- Border Radius: `0.75rem` - Smooth, modern
- Backdrop Filter: `blur(8px)` - Glassmorphism effect

### Typography
- **Main Title**: Golden gradient (`#f6d55c → #d4af37`)
- **Metric Values**: Golden gradient
- **Section Titles**: Bright golden (`#f6d55c`)
- **Labels**: Light golden (`#e2c55f`)
- **Body Text**: Light gray (`rgb(161, 161, 161)`)

### 3D Scene
- **Axis Lines**: Golden (`#d4af37`)
- **Axis Labels**: Golden tones (`#f6d55c`, `#e2c55f`)
- **Grid**: Golden primary lines with dark secondary
- **Background**: Dark gradient (`rgb(10,10,10) → rgb(23,23,23)`)

## Lighting Scheme

```javascript
Ambient: 0.6 intensity, white
Directional: 0.8 intensity, white (clean highlights)
Point Light: 0.3 intensity, golden (warm accents)
Spot Light: 0.4 intensity, bright golden (subtle glow)
```

---

## Material Properties

```javascript
{
  opacity: 0.75,
  emissive: [same as base color],
  emissiveIntensity: 0.2,
  metalness: 0.3,
  roughness: 0.4
}
```

Creates a **premium, slightly metallic look** with subtle glow.
